# Gothic Watch Face 

![Gothic Watch Face  Screenshot](screenshot.png?raw=true "Gothic Watch Face Screenshot")

A minimalist, gothic style Garmin Connect IQ watch face showing:
- Hours & minutes
- Weekday (as astronomical symbol), day of month, numeric month (roman)
- Moon phase
- Maya calendar
- Zodiac
